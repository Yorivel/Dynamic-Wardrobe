# v1.2

## ✨ New Feature: Regional Context Keywords
- **Four context keywords** — `corpo`, `street`, `wild`, `club` — let you assign outfits to broad categories of locations instead of naming every individual area
- **corpo** covers upscale areas: CorpoPlaza, CityCenter, Downtown, NorthOaks, CharterHill, ArasakaWaterfront, JapanTown
- **street** covers gritty neighborhoods: Glen, Wellsprings, Arroyo, LittleChina, Kabuki, Northside, RanchoCoronado, VistaDelRey
- **wild** covers dangerous zones: Badlands, SouthBadlands, NorthBadlands, Pacifica, Coastview, WestWindEstate, Dogtown
- **club** covers nightlife venues: Afterlife, RiotClub, LizziesBar, Clouds, DarkMatter, Totentaz, Embers, ElCoyoteCojo (Heavy Hearts has no location enum and cannot be detected)
- **Multi-keyword scoring** — outfit named `corpo club formal` only applies at clubs in corpo areas (both keywords must match). Higher keyword count = higher priority
- **Customizable mappings** — edit `Core/ContextMap.reds` to add locations, remap areas, or create entirely new context keywords
- **Fallback chain** — specific location match → context keyword match → "outdoor" prefix

## 🐛 Bug Fixes
- **Entrance detection radius tightened** from 5.0 to 1.0 units — reduces the chance of false home exits when loading a save near apartment doors

## 🔧 Technical Details
- **New fallback chain in location callback** — `HasOutfitForLocation` → `HasContextOutfit` (scored, all-must-match) → "outdoor" prefix
- **Context keywords excluded from location matching** — words like "corpo", "street", "wild", "club" in outfit names are filtered out during location word matching to prevent false positives
- **Removed pre-home outfit caching** — leaving an apartment now runs the normal fallback chain (location → context → outdoor) instead of restoring the previously worn outfit
- **`m_wardrobeReady` flag** — guards location callback against firing before `OnGameAttached` finishes initializing
- **Mod-disabled safety** — disabling the mod now only resets in-memory state (no longer clears the persisted quest fact, avoiding race conditions with ModSettings initialization)
