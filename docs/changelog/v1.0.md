# v1.0

## Initial Release
* **Context-driven outfit swapping** — automatically changes Equipment-EX outfits based on game context
* **Apartment detection** — proximity-based entrance detection with enum validation for vanilla apartments (H10, Glen, Northside, Japantown, Corpo Plaza)
* **Home/outdoor outfit cycle** — applies "home" outfit on entry; on exit, matches location, context, or falls back to "outdoor"
* **Scene handling** — intercepts QuestDisable/QuestRestore wardrobe requests to swap nude/home outfits during shower and sleep scenes
* **Ripperdoc support** — detects ripperdoc workspots via Tier4 entry, applies "ripper" outfit, restores previous on exit (designed for use with [Immersive Rippers](https://www.nexusmods.com/cyberpunk2077/mods/7064) and [Immersive Rippers - Dogtown](https://www.nexusmods.com/cyberpunk2077/mods/10255) — vanilla game never uses ripperdoc chairs as active workspots)
* **Drive scene handling** — intercepts UnequipWardrobeSet (EX no-op) to swap outfits during romance drive scenes (designed for use with [Hangout Romances](https://www.nexusmods.com/cyberpunk2077/mods/18972) — the drive scene trigger does not exist in vanilla)
* **Location matching** — bidirectional keyword matching between outfit names and location enums with right-to-left priority
* **Multi-location outfits** — use separators (e.g. `badlands/coastview`) to match multiple areas with a single outfit
* **Random selection** — multiple outfits matching the same keyword are picked at random
* **Unequip blocking** — prevents engine from stripping equipment during Tier4 scene transitions at home
* **Persistent home state** — saves `dw_is_at_home` quest fact so home context survives save/load
* **Shower deferral setting** — optional mode where home outfit applies only after the game naturally undresses V
* **F10 location display** — configurable keybind to show current location enum name on screen
* **Master toggle** — enable/disable all mod features from Mod Settings

## Localization
* **19 languages** — full Codeware localization via ModLocalizationProvider
* Arabic, Chinese (Simplified), Chinese (Traditional), Czech, English, French, German, Hungarian, Italian, Japanese, Korean, Polish, Portuguese (Brazilian), Russian, Spanish, Spanish (Mexican), Thai, Turkish, Ukrainian

## Technical Details
* Event-driven architecture — blackboard listener on `UI_Map.currentLocationEnumName`, no polling
* Equipment-EX integration via `OutfitSystem.GetInstance()` — read-only (never creates or modifies outfit data)
* Single quest fact persistence (`dw_is_at_home`) — safe to uninstall at any time
* Guard against double listener registration in `OnGameAttached`
* `m_wardrobeBlockUnequip` cleared by both WorkspotFinished and QuestRestore (prevents permanent block on abnormal scene exit)
* Cached nude outfit CName on apartment entry for instant scene loading
