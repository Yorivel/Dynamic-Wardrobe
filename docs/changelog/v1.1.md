# v1.1

## ✨ New Feature: Judy's Apartment Support
- **Judy's apartment** (`Kabuki_JudysApartment`) is now recognized as a home location
- **Quest-gated** — only activates after completing Pyramid Song (romance or friendship path)
- **Full home lifecycle** — entrance detection, home/nude/outdoor outfit swapping, save/load persistence
- **Kabuki district fallback** — prevents false home exits on save/load

## 🐛 Bug Fix: Quest Outfit Guard
- **Quest outfit protection** — Dynamic Wardrobe no longer swaps outfits when the game has equipped a quest outfit (e.g. tactical diving suit)
- **Uses Equipment-EX's `IsBlocked()` check** — detects items with the `UnequipBlocked` tag in the Outfit slot
- **All four outfit loading paths guarded** — `ApplyOutfitByPrefix`, `ReapplyOrRandomize`, `ApplyOutfitByLocation`, and `WardrobeLeaveHome`

## 🔧 Technical Details
- **Judy's apartment gated by quest facts** — `sq030_judy_lover >= 1 || sq030_friendship >= 1 || sq030_v_got_apartment >= 1` (covers romance, friendship, and edge case paths)
- **`IsVanillaApartment` renamed to `IsKnownApartment`** — now accepts `player` param for quest system access
- **`IsBlocked()` check** — reuses Equipment-EX's built-in guard that detects the `UnequipBlocked` tag on items in the Outfit slot
