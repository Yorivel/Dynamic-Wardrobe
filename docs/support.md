# Support

## Before Reporting

1. **Check [Known Issues](issues.md)** — your problem might already be documented with a workaround
2. **Update to the latest version** — many issues are fixed in newer releases. See the [Changelog](changelog/README.md) for what's changed
3. **Make sure all requirements are installed** — missing dependencies can cause silent failures. Check the list on the [home page](README.md)
4. **Test with a clean mod setup if possible** — try running Dynamic Wardrobe with only its dependencies to rule out conflicts with other mods

## Reporting a Bug

File bug reports on the [Nexus Mods bugs tab](https://www.nexusmods.com/cyberpunk2077/mods/27791?tab=bugs). Include as much of the following as you can:

* **What happened** and **what you expected** to happen
* **Steps to reproduce** — where were you, what outfit was equipped, what triggered the issue
* **Your outfit names** — the mod relies on keywords in outfit names, so knowing the exact names helps narrow down matching issues
* **Whether the issue persists after reloading** — some problems are one-time glitches that don't survive a save/load cycle

{% hint style="info" %}
Press **F10** (or your configured keybind) in-game to see your current location name. This is useful for reporting location-related issues.
{% endhint %}
