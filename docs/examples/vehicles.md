# Vehicle Examples

## Basic Setup

Start simple — one outfit per vehicle type:

* **`car`** — V's driving look for cars
* **`bike`** — V's riding look for bikes

Or use one outfit for everything:

* **`vehicle`** — any ride, car or bike

## By Context

Different looks for different vibes:

* **`car corpo`** — for luxury rides (Rayfield, Herrera, Chevalier)
* **`car street`** — for muscle and sport cars (Quadra, Mizutani, Archer)
* **`car wild`** — for off-road and nomad trucks (Thorton Mackinaw, Kaukaz)
* **`bike street`** — for city bikes (Arch, Yaiba Kusanagi)

## Targeting Specific Vehicles

Use brand or model names for precision:

* **`car caliburn`** — only when driving a Rayfield Caliburn
* **`car herrera`** — any Herrera (Outlaw or Riptide)
* **`car herrera outlaw`** — only the Herrera Outlaw, not the Riptide
* **`bike kusanagi`** — only the Yaiba Kusanagi
* **`car quadra type66`** — any Quadra Type-66 variant

Press **F10** while driving to see available keywords for the current vehicle.

## With Decorative Names

Extra words that aren't vehicle brands/models are cosmetic:

* **`car casual`** — just a label, works on any car
* **`car corpo executive`** — "executive" is decorative, matches any corpo car
* **`bike leather`** — label, works on any bike

## Multiple Outfits

Same keyword = random pick each time:

* **`car corpo sleek`**, **`car corpo formal`** — random choice on corpo cars
* **`bike wild dusty`**, **`bike wild rugged`** — random choice on wild bikes

## Priority Order

Most specific wins:

1. **`car herrera outlaw`** — exact vehicle (brand + model)
2. **`car herrera`** — brand match
3. **`car corpo`** — context match
4. **`car`** — type match
5. **`vehicle`** — any vehicle
