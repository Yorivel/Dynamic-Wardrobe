# Dynamic Wardrobe

Automatic, context-driven outfit swapping for Cyberpunk 2077 — powered by [Equipment-EX](https://www.nexusmods.com/cyberpunk2077/mods/6945).

Name your outfits with simple keywords. The mod handles the rest.
